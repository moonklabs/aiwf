# Sprint: 생태계 통합 및 완성 (S03)

## Sprint Goal
5개 AI 도구 통합 템플릿을 제공하고, 기본 프로젝트 템플릿을 구축하며, 오프라인 지원과 완전한 문서화로 M02 마일스톤을 완성한다.

## Scope & Key Deliverables

- **Claude Code 템플릿**: `.aiwf/ai-tools/claude-code/`

### 2. 기본 프로젝트 템플릿 3개 (핵심)

### 3. 오프라인 템플릿 캐시 (통합)

- 템플릿 사전 다운로드 메커니즘
- 버전 관리 및 업데이트 전략

### 4. 문서화 및 가이드 완성 (필수)
- M02 전체 기능 사용 가이드
- 설치부터 고급 사용법까지 완전 문서화
