# 지식 프라임

이 프로젝트의 기본 정보로 자신을 준비시키세요.
환경을 인식하고 있는지 확인하세요.

> **참고**: 세션 시작 시에는 `/aiwf:session-start` 명령어를 사용하세요.
> session-start는 prime을 포함한 전체 세션 초기화를 수행합니다.
> 이 명령어는 세션 중간에 단독으로 컨텍스트를 다시 로드할 때 사용합니다.

## **병렬 에이전트**를 사용하여 다음을 수행하세요

- 기본 프로젝트 정보를 위해 .aiwf/01_PROJECT_DOCS/ARCHITECTURE.md 읽기
- 프로젝트 상태를 위해 .aiwf/00_PROJECT_MANIFEST.md 읽기
- 진행 상황 확인을 위해 .aiwf/aiwf-progress.md 읽기 (존재하는 경우)

## 관련 명령어

- `/aiwf:session-start` - 전체 세션 초기화 (prime 포함)
- `/aiwf:session-end` - 세션 종료 및 진행 상황 저장
